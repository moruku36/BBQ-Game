# Sunset BBQ Party

夕焼けの屋外BBQを、参加できなかった友人にも疑似体験してもらうための
ブラウザ向けミニゲームです。  
`HTML / CSS / JavaScript` だけで動作し、焼き台の操作とパーティーの空気感を
1ページで楽しめます。

## 遊び方

1. `index.html` をブラウザで開きます。
2. 焼き台のマスをクリックして選択します。
3. 右側で食材を選び、`網に置く` で焼き始めます。
4. `ひっくり返す` と `お皿にのせる` を使い、友人のリクエストに合う焼き加減で提供します。
5. 火力とにぎやかさを調整しながら、90秒でできるだけ高スコアを目指します。

## 体験できること

- 夕焼け空、ゲスト、火の粉つきのBBQらしい雰囲気
- 牛カルビ、とうもろこし、えび串の焼き分け
- 友人ごとの注文と待ち時間管理
- スコア、コンボ、盛り上がりゲージによるゲーム性
- Canvas を使った1画面完結の軽量な演出

## ファイル構成

- `index.html`: UI 構造
- `styles.css`: レイアウトとビジュアル
- `app.js`: ゲームロジックと Canvas 描画

## メモ

ローカルサーバーは必須ではありませんが、必要なら任意の静的ファイルサーバーでそのまま配信できます。
