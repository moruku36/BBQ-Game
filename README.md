# Sunset BBQ Party

夕焼けのBBQ台で食材を焼くだけの、シンプルなブラウザゲームです。  
`HTML / CSS / JavaScript` だけで動きます。

## 遊び方

1. `index.html` をブラウザで開きます。
2. 食材を選びます。
3. 空いている網をクリックして食材を置きます。
4. 焼けたらその食材をもう一度クリックして回収します。
5. 焦がしすぎる前に回収して高得点を目指します。

## 特徴

- 操作はクリック中心のシンプル設計
- 牛カルビ、とうもろこし、えび串を切り替え可能
- 火力調整あり
- 夕焼けと火の粉のCanvas演出つき

## ファイル構成

- `index.html`: UI
- `styles.css`: 見た目
- `app.js`: ゲームロジックと描画
